应用设置

窗口透明度90%

透明度会作用于工具栏、网页和广告画面。

广告窗口尺寸
×

进入广告模式时使用该尺寸，并自动移动到当前显示器右下角。

老板键

点击后按下新的组合键；必须包含 Ctrl、Alt 或 Shift。注册失败时保留原快捷键。

按下后的动作
关闭按钮行为

同时作用于主窗口和广告画面右上角的关闭按钮。

首页书签

支持 Chrome、Edge、Firefox 等浏览器导出的 HTML；最多导入 100 个 HTTP/HTTPS 书签。内置预设不会被删除。

广告画面

可一次多选 PNG、JPEG 或 WebP；最多 20 张、每张最大 2 MB。图片仅复制到本机应用数据目录。

安全策略

远程网页无法访问 Node 或 Electron API。仅当前可见标签可申请鼠标锁定；摄像头、麦克风、定位、通知和设备权限全部拒绝。
